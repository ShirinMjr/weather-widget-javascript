Weather Widget Project - Static web content

Weather Widget is a simple JavaScript project examining basics of JavaScript language such as action listener, fetch/promises, and DOM Manipulation. A user can enter a location in the box and then application gets the forecast for the selected location. This application makes a call to an api provided by "OpenWeatherMap.org" And displays the result in Celsius and Fahrenheit and an icon representing the weather condition.All you need for this project is a standard browser (Chrome, Firefox...), a code editor.

Before you start, get an API key from Open Weather Map: Our forecast data comes from OpenWeatherMap.org which provides a limited amount of data for free which is great for our project. Requests from this service require an API key which is a unique identifier that associates you with a request coming in. To get your key, you need to create an account on the OpenWeatherMap and get your own unique api key. It should be something like this: e94a50e2e99555c9694b04842d08a572
Follow comments in scriot.js file for better understanding of the project.
